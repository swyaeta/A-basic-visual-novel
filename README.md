# Martin High's Silent Scream

A 2D mystery visual novel built with the Ren'Py engine.

## Story

A high school transfer girl arrives at Martin High looking for a fresh start. But between a bully who makes her life difficult and a mysterious friend whose intentions she can't quite figure out, settling in is harder than she expected.

As she digs deeper into the school's secrets, she realizes that not everything is like it seems. There are several mysteries and hidden truths in the school.

- Ren'Py Visual Novel Engine

## How to Run Locally

1. Download and install Ren'Py from renpy.org
2. Clone or download this repository
3. Open the project in Ren'Py launcher
4. Click Play!
